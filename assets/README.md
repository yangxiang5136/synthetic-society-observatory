## `spatial-orbit-v0.mp4`

Lightweight pre-rendered spatial-video proof for the public viewport. The current file was rendered as a simple 3D orbit scene on the Windows Blender render node, then copied back into this public prototype.

Purpose:

- demonstrate a dynamic observatory viewport without loading WebGL;
- prove that viewpoint changes can map to media anchors instead of local camera rendering;
- keep the public browser in video-decode-only mode.

Companion files:

- `spatial-orbit-v0-poster.jpg`
- `../render-manifests/observatory-spatial-video-v0.json`

Boundary:

- this is a V0 media-consumption demo, not the final 3D scene;
- future assets should be generated by a vendor/cloud render lane or the Windows Blender render node;
- the public page should continue to consume media, not render scene geometry.
